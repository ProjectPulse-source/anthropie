{{ if eq .Lang "en" }} Order is not created. Disorder is displaced. {{ else }} On ne crée pas l'ordre : on déplace le désordre. {{ end }}

{{ if eq .Lang "en" }} Anthropy is the hypothesis that social systems displace disorder rather than resolve it — in space, in time, or between social groups. This site gathers a research corpus devoted to testing this hypothesis: five Anthropy Working Papers, two books, publications and resources designed to be read, cited and discussed. {{ else }} L'anthropie est l'hypothèse selon laquelle les systèmes sociaux ne suppriment pas le désordre : ils le déplacent dans l'espace, dans le temps, ou entre groupes sociaux. Ce site rassemble un corpus de recherche qui la met à l'épreuve : cinq Anthropie Working Papers, deux livres, des publications et des ressources conçues pour être lues, citées et discutées. {{ end }}

{{ if eq .Lang "en" }}Three axes for reading transfers {{ else }}Trois axes pour lire les transferts{{ end }}

{{ if eq .Lang "en" }} Disorder does not disappear. It shifts to another place, another time, or other groups. Anthropy aims to make these displacements visible. {{ else }} Le désordre ne disparaît pas. Il se déplace dans un lieu, dans un futur, ou vers d'autres groupes. L'anthropie propose de rendre ces déplacements visibles. {{ end }}

{{ if eq .Lang "en" }}Spatial{{ else }}Spatial{{ end }}

{{ if eq .Lang "en" }}Centre → periphery{{ else }}Centre → périphérie{{ end }}

{{ if eq .Lang "en" }}Local order is achieved by externalizing costs to other territories: extraction, waste, heat, pollution pushed beyond the visible perimeter.{{ else }}L'ordre local s'acquiert au prix d'une externalisation vers d'autres territoires : métropoles, zones de consommation, centres de décision d'un côté ; extraction, déchets, chaleur, pollutions de l'autre.{{ end }}

{{ if eq .Lang "en" }}Temporal{{ else }}Temporel{{ end }}

{{ if eq .Lang "en" }}Present → future{{ else }}Présent → futur{{ end }}

{{ if eq .Lang "en" }}The present consumes, the future compensates. Public debt, climate debt, nuclear waste and technological debt extend today's costs toward generations that had no say.{{ else }}Le présent consomme, le futur compense. Dette publique, dette climatique, déchets nucléaires ou dette technologique prolongent aujourd'hui leurs coûts vers des générations qui n'ont pas décidé.{{ end }}

Social

{{ if eq .Lang "en" }}Mobile → captive{{ else }}Mobiles → captifs{{ end }}

{{ if eq .Lang "en" }}Those with financial, legal or residential mobility transfer burdens more easily toward those who cannot refuse them: users, precarious workers, taxpayers, subcontracted labour, relegated groups.{{ else }}Ceux qui disposent de mobilité financière, juridique ou résidentielle transfèrent plus facilement les charges vers ceux qui ne peuvent pas les refuser : usagers, précaires, contribuables, travailleurs sous-traités, groupes relégués.{{ end }}

{{ if eq .Lang "en" }}The corpus{{ else }}Le corpus{{ end }}

{{ if eq .Lang "en" }} The anthropic framework extends across several formats: working papers, books, reference pages and publications in the intellectual press. The aim is not to multiply formats but to make the same mechanism visible through different objects — debt, energy, deep history, epistemic order. {{ else }} Le cadre anthropique se déploie sur plusieurs supports : working papers, livres, pages de référence et publications dans la presse intellectuelle. L'enjeu n'est pas de multiplier les formats, mais de rendre visible un même mécanisme à travers des objets différents — dette, énergie, longue durée, ordre épistémique. {{ end }}
